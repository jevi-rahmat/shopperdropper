# ShopperDropper
E-commerce website developed using PHP, MySQL, and BootStrap.

### Features
- Authentication
Implemented login and register feature for role customer and supplier.

- Browse
Implemented search product feature to enable users to search for specific products.

- Account
Implemented account dashboard to monitor user's activity. It also support the features for updating and deleting account.

- Monetary
Implemented dummy logic to simulate transactions. Support the top-up feature with dummy data.

### Roles

- Customer
Customer can view products, make purchases with dummy data, top-up with dummy data, and view transaction history.

- Supplier
Supplier can view products, add products, and monitor their products with update and delete features.

- Administrator
Administrator can view products, manage users (Customer and Supplier), manage monetary features, and manage products.
